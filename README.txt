To be able to use d3.csv function for importing csv files the application needs to be run on http server. Several options described here:
https://github.com/processing/p5.js/wiki/Local-server

Example:

Run in terminal(not working on windows)

python -m SimpleHTTPServer 8888 &

This sets up a local http server. Go to http://localhost:8888/ to "launch" application. index.html is the app file that is run here.
